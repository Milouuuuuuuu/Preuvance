# Northstar — fictional human-review procedure

Status: demonstration document only. Not an approved policy and not evidence for a real organisation.

1. The support copilot may produce a draft only from the approved demo knowledge base.
2. A support agent reads the cited source and the complete draft before sending.
3. The agent edits or rejects any unsupported statement.
4. The demo owner records the application version used for the review.
5. A monthly sample is inspected for missing citations and inappropriate disclosure.

This file exists to demonstrate local SHA-256 calculation, source metadata and review fields in Preuvance. Its content must not be presented as a certified or production control.
